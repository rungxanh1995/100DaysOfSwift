# Whack-a-penguin
## About
Whack-a-penguin is a game that mimics your childhood fav "Whack-a-mole", letting you whack a penguin emerging from a ground hole 🐧🕹.

Game best played with sound on 🔊

## Concepts Utilized
* `SpriteKit`
* `Subclasses of SKNode`
* `Code Encapsulation`
